# abdulmuhaymin.github.io
## About 
This is a simple single webpage which includes links to all the other webpages I host through github. Currently four different repositories are linked in this webpage: Abstract, Thoughts, Symmetry-Conservations-Unification, and Pseudocode.
## Details
**Abstract:** A blog on mathematics and the relation between mathematics and the physical world. 

**Thoughts:** A blog on philosophy, mostly concerned with logic, metaphysics, paradoxes and contemporary issues.

**Symmetry-Conservations-Unification:** A blog mostly focused on theoretical physics and physicists

**Pseudocode:** A blog on computer programming, algorithm, computational physics and what not! The main focus of this blog is high performance programming using Julia.
